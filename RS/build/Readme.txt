===============================================================================
| This directory will contain the files generated during the front-end compilation.
===============================================================================

===============================================================================
| In this directory will be generated the subdirs:
| -----------------------------------------------------------------------------
| * moc_files: It will contain the source-codes (.cpp) generated from the meta-
|              -objects of the front-end, that are defined in the Control and View.
| -----------------------------------------------------------------------------
| * objs     : It will contain the object files (.o) of the front-end compilation.
| -----------------------------------------------------------------------------
| * res_files: It will contain the source-codes (.cpp) of the front-end resources
|              files. These sources have the binary serialization from the
|              multimedia resources of the front-end (icons, images, sounds, ...)
|              to be embedded in the executable.
| -----------------------------------------------------------------------------
| * ui_files : It will contain the source headers (.h) generated from the graphical
|              forms (.ui).
===============================================================================

===============================================================================
| NOTE: It is not necessary change anything file generated, once these are
| generated during the compilation process.
| However, if some update in the program source don't have effect, clear (exclude)
| all the subdirs, thereby ensure that the files will be (re)generated in the next
| compilation.
===============================================================================
